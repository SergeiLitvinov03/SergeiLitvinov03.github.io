As training software, SYMplus with plusCARE™ Milling supports quick familiarization with DIN programming according to PAL and Siemens. Above all, SYMplus with plusCARE™ is a CAD/CAM system that is particularly easy to learn, with which you can save programming time, avoid crashes, reduce production time and also generate NC programs for different OPTIMUM machines or Siemens controls (802S, 808D, 828D, 840D, ...) with a uniform interface. System requirements for the plus systems:
  • Supported operating systems: Microsoft Windows® 7/8/10 (32/64 bit)
  • Screen resolution min. 1024 x 768
  • OpenGL-compatible 3D graphics card, e.g. GeForce GT 210 (1024 MB)
  • Main memory: min. 2 GB
  • Approx. 2 GB free hard disk space per technology for system data
